Sobre
---
DemoSunmiPrinter é um aplicativo Android feito em Flutter que utiliza o serviço de impressão SunmiPrinter.</br>

 Funcionalidades
---

- [X] Impressão de imagem 
- [X] Impressão de Texto
- [X] Impressão de QrCode
- [X] Impressão de Código Barras
- [X] Pular linha
- [X] Cortar papel

Getting Started
---

Executando projeto 

```bash
flutter run
```
